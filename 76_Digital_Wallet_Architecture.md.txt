2. CORE PRINCIPLES
2.1 Wallet Is Not The Ledger
Wallet provides user experience.
Ledger remains the financial source of truth.

Le Wallet ne remplace jamais le Ledger.

2.2 Instant Financial Experience
Wallet operations must feel immediate.

Les utilisateurs doivent voir leurs opérations en temps réel.

2.3 Security By Default
Every wallet action must be authenticated and protected.

27. CLAUDE CODE RULES

Claude Code must:

NEVER use wallet balance as financial truth.
ALWAYS integrate wallet operations with Ledger.
ALWAYS protect wallet credentials.
ALWAYS implement transaction security.
NEVER bypass KYC requirements.
ALWAYS maintain wallet transaction history.
ALWAYS support multi-currency design.
ALWAYS validate wallet limits.
ALWAYS implement fraud controls.
ALWAYS follow Enterprise Wallet Architecture Standards.
28. SUCCESS CRITERIA

Le Digital Wallet Amanah est conforme lorsque :

les clients peuvent gérer leur argent instantanément ;
toutes les opérations sont enregistrées dans le Ledger ;
les paiements sont sécurisés ;
les soldes restent exacts ;
les intégrations externes sont possibles.

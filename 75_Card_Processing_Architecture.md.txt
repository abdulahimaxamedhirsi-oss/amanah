2. CORE PRINCIPLES
2.1 Card Security First
Card data must always be protected.

Les données cartes sont considérées comme hautement sensibles.

2.2 Real-Time Authorization
Every card transaction must receive a fast and reliable decision.
2.3 Tokenization Everywhere
Sensitive card information should be replaced with secure tokens.

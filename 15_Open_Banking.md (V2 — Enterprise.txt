15_Open_Banking.md (V2 — Enterprise Financial Connectivity Layer)
1. OBJECTIF DU SYSTÈME

Ce module définit comment Amanah se connecte au monde extérieur :

banques, mobile money, cartes, fintechs partenaires, rails de paiement

Rôle principal
connecter Amanah aux systèmes bancaires externes
intégrer mobile money (Afrique)
gérer M-Amanah plugin (agents physiques)
gérer les confirmations de settlement
normaliser tous les partenaires financiers
2. CORE PRINCIPLE
2.1 Amanah is NOT a bank
Amanah = financial orchestration layer
NOT a bank holding funds directly
2.2 External rails abstraction

Tous les systèmes externes doivent être standardisés via :

connectors
adapters
unified API layer
2.3 Settlement-first model
internal ledger = truth
external systems = execution layer

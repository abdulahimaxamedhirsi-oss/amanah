La Data Governance Platform est responsable de la qualité, de la gouvernance, de la sécurité et du cycle de vie de toutes les données d'Amanah.

21. CLAUDE CODE RULES

Claude Code must:

ALWAYS use the Master Data Management system as the authoritative source.
NEVER duplicate business entities across services.
ALWAYS classify sensitive data before storage.
ALWAYS enforce data retention policies.
NEVER expose restricted data to unauthorized users.
ALWAYS maintain complete data lineage.
ALWAYS validate data quality before persistence.
ALWAYS encrypt sensitive information at rest and in transit.
NEVER bypass the Data Governance Platform.
ALWAYS keep metadata synchronized with schema changes.
22. SUCCESS CRITERIA

La Data Governance Platform est considérée comme complète lorsque :

toutes les données sont gouvernées de manière centralisée ;
les entités critiques disposent d'une source unique de vérité (MDM) ;
la qualité, la confidentialité et la traçabilité sont garanties ;
les politiques de conservation et de confidentialité sont appliquées automatiquement ;
les données sont exploitables de manière fiable pour l'analytique, l'IA et les obligations réglementaires.

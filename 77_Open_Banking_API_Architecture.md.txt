2. CORE PRINCIPLES
2.1 API First Banking
Every banking capability must be available through secure APIs.

Toutes les fonctionnalités bancaires doivent être accessibles via des APIs documentées.

2.2 Customer Owns The Data
Customers control who can access their financial data.

Le consentement du client est obligatoire.

2.3 Zero Trust API Security
Every API request must be authenticated, authorized and audited.

(V3 — Enterprise AML, Transaction Monitoring, Sanctions Screening & Financial Crime Compliance Framework)
1. OBJECTIF

Ce document définit le framework officiel AML d'Amanah.

Il couvre :

Anti-Money Laundering (AML) ;
Counter-Terrorist Financing (CTF) ;
Transaction Monitoring ;
Customer Risk Rating ;
Sanctions Screening ;
PEP Screening ;
Case Management ;
Suspicious Activity Reporting (SAR/STR) ;
Financial Crime Intelligence ;
Continuous Monitoring.

L'objectif est de prévenir, détecter et signaler les activités financières suspectes conformément aux réglementations internationales.

2. CORE PRINCIPLES
2.1 Risk-Based Approach

AML controls shall be proportional to customer and transaction risk.

2.2 Real-Time Detection

High-risk transactions must be detected before completion whenever possible.

2.3 Continuous Compliance

AML controls remain active throughout the customer lifecycle.

26. CLAUDE CODE RULES

Claude Code must:

ALWAYS evaluate transactions against AML rules before completion whenever technically feasible.
NEVER allow sanctioned parties to transact without following applicable legal and regulatory requirements.
ALWAYS maintain customer AML risk scores.
ALWAYS preserve immutable AML audit trails.
ALWAYS support configurable rule-based and AI-assisted detection.
ALWAYS integrate sanctions, PEP and adverse media screening.
ALWAYS maintain complete investigation histories.
ALWAYS support SAR/STR generation and regulatory reporting.
ALWAYS separate AML decision-making from operational transaction processing through appropriate governance.
ALWAYS follow Amanah Enterprise AML Standards.
27. SUCCESS CRITERIA

Le framework AML est conforme lorsque :

les transactions sont surveillées en temps réel ;
les clients sont évalués selon une approche basée sur le risque ;
les alertes sont investiguées de manière structurée ;
les déclarations réglementaires sont produites conformément aux exigences légales ;
chaque décision AML est entièrement traçable et auditée.
